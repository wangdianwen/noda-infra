**BUG 1 + BUG 2 修复（第 136、186、239 行）：将 `rclone config create` 替换为直接写入配置文件**

```bash
cat > "$rclone_config" <<EOF
[b2remote]
type = b2
account = $b2_account_id
key = $b2_application_key
EOF
```

- `test_b2_operations()` 第 237-241 行：同上处理
