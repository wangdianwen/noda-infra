## 🚀 部署架构

### 容器化部署

**容器名称**: opdev  
**镜像**: noda-backup:latest  
**网络**: noda-network（noda-infra 分组）  
**重启策略**: unless-stopped

**集成服务**:
- PostgreSQL（noda-infra-postgres-1）
- Nginx（noda-infra-nginx-1）
- Keycloak（noda-infra-keycloak-1）

### 定时任务配置

```cron
# 每天凌晨 3:00 执行备份
0 3 * * * /app/backup-postgres.sh

# 每周日凌晨 3:00 执行验证测试
0 3 * * 0 /app/test-verify-weekly.sh

# 每 6 小时清理历史记录
0 */6 * * * /app/lib/metrics.sh cleanup

# 每天凌晨 4:00 清理旧备份（7 天前）
0 4 * * * find /tmp/postgres_backups -type f -name "*.dump" -mtime +7 -delete
```

### 持久化存储

```
docker/volumes/
├── backup/      # 备份文件存储
├── history/     # 历史记录存储
└── logs/        # 日志文件存储
```
