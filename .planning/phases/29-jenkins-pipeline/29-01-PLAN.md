**新增函数 2: pipeline_backup_database**
部署前自动备份（per D-06）。
参数: $1 = SERVICE
逻辑:
- backup_dir="${BACKUP_HOST_DIR:-$PROJECT_ROOT/docker/volumes/backup}/infra-pipeline/${service}"
- timestamp=$(date +"%Y%m%d-%H%M%S")
- backup_file="${backup_dir}/${timestamp}.sql.gz"
- mkdir -p "$backup_dir"
- 对 keycloak: `docker exec noda-infra-postgres-prod pg_dump -U postgres --clean --if-exists keycloak | gzip > "$backup_file"`
- 对 postgres: `docker exec noda-infra-postgres-prod pg_dumpall -U postgres --clean --if-exists | gzip > "$backup_file"`
- 对 nginx/noda-ops: log_info "$service 不需要备份（无持久化数据）" 并 return 0
- 验证文件大小 > 1024 字节，使用 `stat -f%z "$backup_file" 2>/dev/null || stat -c%s "$backup_file"` (macOS/Linux 兼容)
- 备份失败: log_error + return 1
- 成功: log_success + export INFRA_BACKUP_FILE="$backup_file"
