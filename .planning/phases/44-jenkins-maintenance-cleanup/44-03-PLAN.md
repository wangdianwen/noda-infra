**步骤 1: 在 Jenkins 中注册 cleanup Pipeline（如果尚未注册）**

1. 浏览器访问 `http://<server-ip>:8888`
2. 点击 "New Item"
3. 输入名称: `cleanup`
4. 选择类型: "Pipeline"
5. 点击 OK
6. 在配置页面中：
   - Pipeline -> Definition: 选择 "Pipeline script from SCM"
   - SCM: 选择 "Git"
   - Repository URL: 填写项目仓库路径
   - Branch Specifier: `*/main`
   - Script Path: `jenkins/Jenkinsfile.cleanup`
7. 点击 Save

**步骤 2: 首次构建（注册参数）**

1. 点击 "Build Now"（首次构建 parameters 块不会生效，这是正常的）
2. 等待构建完成，确认 Pipeline 阶段视图显示 5 个阶段

**步骤 3: 带 FORCE 参数的第二次构建**

1. 点击 "Build with Parameters"
2. 勾选 FORCE = true（强制执行 pnpm store prune，跳过 7 天间隔）
3. 点击 "Build"
4. 等待构建完成

**步骤 4: 验证构建日志**

检查第二次构建日志，确认以下输出：
- "磁盘快照: 定期清理前" -- 磁盘快照（清理前）

- "pnpm store prune 完成" -- pnpm store 清理成功
- "npm cache 清理完成" -- npm cache 清理成功
- "磁盘快照: 定期清理后" -- 磁盘快照（清理后）
